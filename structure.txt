build  -----存放构建生成的目标文件
data   ------存放移动对象仿真请求数据
design ------存放项目设计文件,包括类图
src    ------源文件目录
    base ------ 基础功能的封装类，包括线程、日志、配置文件等
    tool ------ 外部工具，生成和查看仿真数据
    core ------ smart的核心处理逻辑目录
        center ------ 存放全局配置类和外观类
        request ------ 请求相关的类
        thread  ------ 线程相关的类
        store   ------ 基本的存储结构
        index   ------ 索引相关的类
            PGrid   ------ 别人的并行处理索引
            ST2Btree ----- 别人的自适应索引
            ST2Grid  ----- 我们的综合索引
